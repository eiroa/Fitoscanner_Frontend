#Aplicación Frontend del Fitoscanner

El siguiente proyecto constituye uno de los 3 componentes del Trabajo de Inserción Profesional (TIP) de Mauro Adolfo Eiroa, para completar la carrera de Tecnicatura en Programación Informática.

El objetivo de este proyecto es proveer una aplicación web para utilizarse en cualqueir navegador actual, contiene toda la lógica y las pantallas necesarias para administrar toda la aplicación Backend, ya sea cargar nuevas especies, tratamientos, y resolver muestras de usuarios manualmente. Este proyecto fue pensado originalmente para ser puesto en producción como una aplicación Java en un servidor web, sin embargo, dada la naturaleza de la aplicación, un servidor dedicado para este proyecto solamente tiene un rol de contenedor del frontend, puesto que la ejecución se realiza en los dispositivos de los clientes una vez descargada. Por tal razón el código de este proyecto puede ser hosteado en cualquier tipo de servidor.

Los dos proyectos adicionales que completan el TIP, son la aplicación Backend, que posee toda la lógica necesaria para resolver las muestras de los usuarios, y la aplicación Android, orientada exclusivamente a los usuarios finales para obtener las muestras de malezas.

Tanto este proyecto como los dos restantes son liberados a la comunidad bajo licencia GPL V3
